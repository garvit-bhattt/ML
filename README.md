# Machine Learning (ML) Concepts & Intuition 🧠

Welcome to the **ML** repository! This project is a curated collection of Jupyter Notebooks and resources designed to build a strong foundation in Machine Learning, starting from the core mathematical and probabilistic "intuition" required for data science.

## 📁 Repository Structure

| File | Description |
| :--- | :--- |
| [Prob_Distributions_Intuition.ipynb](./Prob_Distributions_Intuition.ipynb) | Deep dive into Random Variables (RV), Probability Mass Functions (PMF), and Probability Density Functions (PDF) with Python visualizations. |
| [Common_Probability_Distributions.ipynb](./Common_Probability_Distributions.ipynb) | Explorations of Bernoulli, Binomial, Exponential, and Gaussian distributions. |
| [Histograms_and_KDE.ipynb](./Histograms_and_KDE.ipynb) | Understanding data shapes using Histograms vs. Kernel Density Estimation (KDE). |
| [Gradient_Descent_Intuition.ipynb](./Gradient_Descent_Intuition.ipynb) | Intuition and scratch implementation of Gradient Descent optimization. |

## 🚀 Getting Started

### Prerequisites

To run the notebooks in this repository, you'll need Python installed along with the following libraries:

- `numpy`
- `matplotlib`
- `scipy`
- `seaborn`

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/garvit-bhattt/ML.git
   cd ML
   ```

2. **Install dependencies:**
   ```bash
   pip install numpy matplotlib scipy seaborn
   ```

3. **Launch Jupyter:**
   ```bash
   jupyter notebook
   ```

## 📖 Topics Covered

This repository aims to cover:
1. **Probability & Statistics:** Random Variables, PMF/PDF, Normal Distribution, Bayes' Theorem.
2. **Data Visualization:** Histograms, KDE, Box Plots, Scatter Plots.
3. **Optimization Algorithms:** Gradient Descent, SGD, Momentum, Adam.
4. **Linear Algebra:** Vectors, Matrices, Eigenvalues/Eigenvectors.
5. **Supervised Learning:** Regression, Classification.
6. **Unsupervised Learning:** Clustering, Dimensionality Reduction.
